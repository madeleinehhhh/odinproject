Learning Outcomes

  • What are the eight data types of javascript?
    1. number
    2. BigInt -- for numbers bigger than 2^53 - 1. a value is a BigInt if we add n to the end, e.g. 1234567890123456789012345678901234567890n
    3. strings, denoted with simple quotes, single or double, or extended functionality backticks: ', ", `
      you can embed variables and expressions into a backtick string by putting them in: ${...}.
    4. boolean
    5. null -- would be assigned to something with an unknown or undefined value, like if someone hasn't input their age yet
    6. undefined -- could but shouldn't be assigned to a variable, and is the default initial value until another variable is assigned
    7. objects -- can store more complex collections of data than the other "primitive" types which contain only a single thing (a string, number, etc.)
    8. symbols... create unique identifiers for objects... more later

  • What is the difference between slice/substring/substr?
    quoting from w3 article on string methods
    • slice() extracts a part of a string and returns the extracted part in a new string.
    • substring() cannot accept negative indexes -- i.e. you can't slice from the back
    • substr() is similar to slice(). The difference is that the second parameter specifies the length of the extracted part.

  • What are the three logical operators and what do they stand for?
    • OR: ||
    • AND: &&
    • NOT: !
    • and it appears there is a fourth... Nullish Coalescing: ??
    you can use && and || to give you one of multiple comparisons. AND returns the first falsy value while OR returns the first truthy one.
    !! converts anything to its boolean. !! is the same as Boolean().
    in order of precedence, ! then && then ||

      == checks equality without checking types, whereas === checks for type first.
    • not equals: a != b
